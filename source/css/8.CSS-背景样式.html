1.css和背景相关的属性

常见属性： background-color: #fff;设置元素的背景(前景色)颜色。 background-image: url("路径地址");用来设置元素的背景图片，图片较大只显示左上角，一样大全部显示，小了则会默认平铺充满整个元素，元素没设置宽高显示不出来的。
background-repeat: no-repeat;设置背景图片小于元素时如何显示，默认repeat平铺充满元素，no-repeat不平铺只显示一张，repeat-x水平方向平铺，repeat-y垂直方向平铺。 background-size: auto;设置背景图片的大小，auto就是以原始图大小显示。cover拉伸图片覆盖元素。或者具体值。 background-position: 0 0;第一个值水平方向 left，center，right，第二个值垂直方向top center bottom。只设置一个另一个默认为center。 background-attachment: fixed;用来设置背景图片是否随着浏览器窗口滚动。fixed固定，scroll滚动默认值 background:url() color no-repeat 0 0;缩写属性，没有顺序要求，也没有数量要求。不写就使用默认值。
background-image和img元素的区别：

background-image是一个css的属性 img是一个HTML元素

2.css-sprite精灵图

css-sprite是一种css图像合成技术，将各种小图片合并到一张图里，然后利用css的background-position属性来定位想要的那个小图片。 好处：减少网页的http请求次数，加快网页响应的速度，减少服务器的压力,减少图片的总大小。 制作：使用ps
