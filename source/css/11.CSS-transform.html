css的transform(形变)属性

transform属性允许平移，旋转，缩放，倾斜指定的元素。常见的函数有四个
translate(x,y),元素往x轴和y轴平移设置的大小，百分比值时相对自己本身大小
rotate(deg),元素旋转多少deg(度degree)，正数顺时针，负数逆时针
scale(x,y),元素往x轴和y轴缩放,一个值x和y轴相同的缩放，1保持不变，2放大一倍，0.5缩小一半。
skew(deg,deg),元素往x轴和y轴倾斜,正数顺时针，负数逆时针

translate(x,y),平移元素
