4. <div>，<span>标签：

<div>，<span>两个标签都没有语义，但是都是一个容器，用来装其它元素形成一个整体便于操作。
其中div可以装其它任何元素，且一直会用。 span元素用来装少量文本，且这些文本需要设置特殊的样式时就可以考虑使用。

div元素是一个块元素，独占一行，用来布局(作为一个父容器把其它元素包括起来)。
span元素是一个行内元素，不独占一行也是用来布局(区分普通文本)，我是第二个span不换行

5. <img>图片标签：

行内替换元素与前后的文字处在同一行，用来在网页上插入图片默认以原始大小显示，是一个单标签。 一般img会放在a标签内部，使得图片也变成一个可以点击的超链接。 常用属性：
src：source缩写引入图片资源的地址路径(远程或本地地址)
alt：设置图片的描述信息(图片不能正常显示时显示的文字信息)
width：设置宽度，只设置一个时另一个会等比例缩放的，但是一般在css中设置。
height：设置高度
loading属性可以指定图片的懒加载，即图片默认不加载，只有即将滚动进入视口才加载。 auto：浏览器默认行为，等同于不使用loading属性。 lazy：启用懒加载。 eager：立即加载资源，无论它在页面上的哪个位置。 注意：它是行内替换元素所以可以设置宽高，单位是像素或百分比。 路径问题：
绝对路径：计算机上一层一层的。
相对路径：相对于当前文件的路径， ./ 表示当前文件，../ 表示当前文件夹的上一层文件夹
图片格式问题：
Jpeg(jpg) 图片支持颜色比较多，可以压缩不透明，一般用来保存照片。
gif 支持颜色少，简单透明，支持动态图 png 静态图片，透明。

6. <a>超链接标签：

超链接是一个行内元素，anchor錨的缩写，作用是允许用户在页面上，从一个网址跳转到另一个网址，从而把所有资源联系在一起。
注意：a元素可以包含任意其它元素除了本身。
属性：
href：链接跳转指向的网址,它的值应该是一个 URL 或者锚点。是锚点时点击后，浏览器会自动滚动到锚点所在的位置。
title：属性给出链接的说明信息。即鼠标悬停在链接上方时，浏览器会将这个属性的值，以提示块的形式显示出来。 target：用来指定打开跳转链接窗口的方式， 值_self(默认)，在当前窗口中打开， 值_blank在新的窗口视图中打开。

6.1拓展知识：URL 简介

URL 是“统一资源定位符”（Uniform Resource Locator）的首字母缩写，中文译为“网址”，表示各种资源的互联网地址。 所谓资源，可以简单理解成各种可以通过互联网访问的文件，比如网页、图像、音频、视频、JavaScript 脚本等等。 只要有它的网址就能获取到。 URL的组成： 如下 https://www.example.com:80/path/to/myfile.html?key1=value1&key2=value2#anchor 协议://主机(域名):端口号/路径?querystring#anchor 协议（scheme）是浏览器请求服务器资源的方法 主机（host）是资源所在的网站名或服务器的名字，又称为域名。 端口（port）同一个域名下面可能同时包含多个网站，它们之间通过端口（port）区分。 路径（path）是资源在网站的位置。 查询参数（parameter）是提供给服务器的额外信息。参数的位置是在路径后面，两者之间使用?分隔 锚点（anchor）是网页内部的定位点，使用#加上锚点名称，放在网址的最后
